EnKore
======

EnKore is a multilingual keyboard, supporting automatic keboard management when typing in English and Korean.


Instruction
-----------

** Everything in PYTHON3! HEHE**
* Before running, in the console:
```
$ export PYTHONIOENCODING=utf8 
```
